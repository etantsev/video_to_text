# Text from Video

Небольшой проект для извлечения аудиодорожки из видео и преобразования речи в текст.

## Что делает проект

- Берёт видеофайл `VID.mp4`.
- Извлекает аудио в `audio.wav` с помощью `ffmpeg`.
- Распознаёт речь через `faster-whisper`.
- Сохраняет результат в:
  - `transcript.txt` — обычный текст;
  - `transcript.srt` — субтитры с таймкодами.

## Требования

- Python 3
- `ffmpeg`

Проверить наличие `ffmpeg` можно командой:

```bash
ffmpeg -version
```

## Установка

Создайте виртуальное окружение и установите зависимости:

```bash
python3 -m venv .venv
.venv/bin/pip install faster-whisper
```

## Использование

1. Поместите видеофайл в каталог проекта и назовите его `VID.mp4`.
2. Извлеките аудио:

```bash
ffmpeg -y -i VID.mp4 -vn -acodec pcm_s16le -ar 16000 -ac 1 audio.wav
```

3. Запустите распознавание:

```bash
.venv/bin/python transcribe.py
```

После выполнения появятся файлы `transcript.txt` и `transcript.srt`.

## Настройки

В `transcribe.py` можно изменить:

- `AUDIO_PATH` — путь к аудиофайлу;
- `MODEL_SIZE` — размер модели Whisper, например `tiny`, `base`, `small`, `medium`, `large-v3`.

Чем больше модель, тем выше качество распознавания, но тем дольше обработка и выше требования к ресурсам.
